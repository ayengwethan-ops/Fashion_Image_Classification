# Fashion Image Classification

## Project Overview

This project is a Deep Learning based Fashion Image Classification system that classifies clothing images into different fashion categories. The project implements a **multi-class image classification** model using Convolutional Neural Networks (CNN).

Users can upload a fashion image through the web application, and the trained model predicts the clothing category.

---

## Problem Statement

The goal of this project is to automatically identify and classify fashion products from images. This helps demonstrate the practical application of computer vision and deep learning in the fashion industry.

This project uses **multi-categorical image classification** because the dataset contains multiple clothing categories instead of just two classes.

---

## Dataset Information

- Dataset Type: Fashion Image Dataset
- Classification Type: Multi-Class Classification
- Number of Classes: Multiple fashion categories
- Image Type: Clothing/Fashion product images
